Design System

Linear meets Obsidian — Andrew Fraser Dashboard Suite

Design Philosophy

Opacity, Not Colors

Build palette from white/black at different opacities over the base. Everything mathematically related.

Color = Meaning

Reserve color for semantic use only: status, priority, urgency. Never decorative.

Whisper, Don't Shout

Subtle hover states. Low-opacity accent washes. Fast transitions. Nothing screams.

Earn Every Pixel

If an element doesn't help the user do something, remove it. Content-first, always.
